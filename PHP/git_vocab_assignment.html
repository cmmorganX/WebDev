Git Terminology and Definitions

What is version control software?

It is in charge of tracking every change to the code in a certain database. It's like a backup and a log of all the changes that have been made.

What is Git Add Command?

It is a command for when you want to add something or change something to the project you're working on but it doesn't actually change anything yet. That's where commit comes in.

What is Git Commit Command?

It's for saving your changes.

What is Git Push Command?

It's for uploading files from your local machine to your online repository.

What is Git Pull Command?

This command downloads content from an online repository to the local repository and forces the local to match the online one.

What is Git Clone Command?

Creating a copy of a specific repository.
